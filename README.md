## Stack

| Layer | Technology |
|---|---|
| Framework | React 18 + TypeScript |
| Build | Vite |
| Styling | Tailwind CSS |
| Icons | Phosphor Icons |
| Database | Firebase Firestore + Storage |
| Forms | Formspree |
| Markdown | react-markdown + rehype-highlight |
| Deployment | GitHub Pages |

---

## Features

- **Hero** — parallax background, cursor glow, staggered entrance animations, animated scroll indicator
- **About** — Current role card, education timeline, experience section
- **Skills** — filterable skill grid with magnetic hover effect and stagger animations
- **Projects** — expandable list with tech stack, features, image previews, README viewer
- **Blogs** — card grid with per-category accent colours and hover interactions
- **Testimonials** — paginated 2×2 grid with avatar initials
- **Contact** — floating label form with honeypot, phone input, animated success state
- **Resume** — in-browser PDF viewer with copy link + share + download
- **Private Vault** — Firebase-backed notes, links, and file storage behind auth
- Fully responsive across all screen sizes
- Scroll-triggered reveal animations with `prefers-reduced-motion` support
